Detailed Description

template<std::size_t MaxElements, typename T>
class cljonic::Repeat< MaxElements, T >

The Repeat type is a fundamental immutable collection type in cljonic. It is implemented as a lazy sequence of a specified length of a specified value. Many Core functions accept Repeat arguments.

template<std::size_t MaxElements, typename T >
using cljonic::Repeat< MaxElements, T >::cljonic_collection_type = std::integral_constant<CljonicCollectionType, CljonicCollectionType::Repeat>

There are two ways to create a Repeat:

  • Repeat{value} returns a Repeat of value std::numeric_limits<std::size_t>::max() times
  • Repeat<count, valueType>{value} returns a Repeat of value, which is of type valueType, count times

Note: The Repeat type, unlike Array and Set, specifies the number of elements before the type. This is consistent with Clojure where something like (repeat 10 "Hello")< > is used to create a repeating sequence of ten instances of the value "Hello".

#include "cljonic.hpp"
using namespace cljonic;
int main()
{
using T = std::variant<int, double, char, const char*>;
const auto r0{Repeat{1}}; // immutable, 1, std::numeric_limits<std::size_t>::max() times
const auto r1{Repeat<10, int>{1}}; // immutable, 1, ten times
const auto r2{Repeat{T{'x'}}}; // immutable, T{'x'}, std::numeric_limits<std::size_t>::max() times
const auto r3{Repeat<100, T>{T{'x'}}}; // immutable, T{'x'}, 100 times
return 0;
}
