Detailed Description

template<typename T>
class cljonic::Repeat< T >

The Repeat type is a fundamental immutable collection type in cljonic. It is implemented as a lazy sequence of repeating values of a specified type. Many Core functions accept Repeat arguments.

◆ cljonic_collection_type

template<typename T >
using cljonic::Repeat< T >::cljonic_collection_type = std::integral_constant<CljonicCollectionType, CljonicCollectionType::Repeat>

There are two Repeat constructors:

  • Repeat(value) returns a Repeat of value std::numeric_limits<std::size_t>::max() times
  • Repeat(count, value) returns a Repeat of value count times
    #include "cljonic.hpp"
    using namespace cljonic;
    int main()
    {
    using T = std::variant<int, double, char, const char*>;
    const auto r0{Repeat(1)}; // immutable, 1, std::numeric_limits<std::size_t>::max() times
    const auto r1{Repeat(10, 1)}; // immutable, 1, ten times
    const auto r2{Repeat(T{'x'})}; // immutable, T{'x'}, std::numeric_limits<std::size_t>::max() times
    const auto r3{Repeat(100, T{'x'})}; // immutable, T{'x'}, 100 times
    return 0;
    }
