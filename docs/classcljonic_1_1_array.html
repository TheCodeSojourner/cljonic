Detailed Description

template<typename T, std::size_t MaxElements>
class cljonic::Array< T, MaxElements >

The Array type is a fundamental immutable collection type in cljonic. It is implemented as a C array, and does not use heap memory. An Array has a specified maximum number of ordered elements each of the same specified type (i.e., It is homogenous). Note that one could create an Array of a UNION or std::variant to get something like heterogeneity. Many Core functions accept Array arguments.

◆ cljonic_collection

template<typename T , std::size_t MaxElements>
using cljonic::Array< T, MaxElements >::cljonic_collection = std::true_type

The Array constructor returns an instance of Array initialized with its arguments. If the number of arguments exceeds the maximum number of elements specified, the extra arguments are silently ignored.

#include "cljonic.hpp"
using namespace cljonic;
int main()
{
const auto a0{Array<int, 10>{}}; // immutable and empty
const auto a1{Array<int, 10>{1, 2, 3, 4}}; // immutable and sparse
const auto a2{Array<int, 4>{1, 2, 3, 4}}; // immutable and full
const auto a3{Array<int, 4>{1, 2, 3, 4, 5, 6}}; // immutable and full, and the values 5 and 6 are ignored
const auto a4{Array{1, 2, 3, 4}}; // immutable and full of four int values
return 0;
}
