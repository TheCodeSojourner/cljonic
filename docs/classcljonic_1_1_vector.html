Detailed Description

template<typename T, std::size_t MaxElements>
class cljonic::Vector< T, MaxElements >

The Vector type is a fundamental collection type in cljonic. It is implemented as a C array, and does not use heap memory. A Vector has a specified maximum number of ordered elements each of the same specified type (i.e., It is homogenous). Note that one could create a Vector of a UNION or std::variant to get something like heterogeneity. Many Core functions accept Vector arguments.

◆ cljonic_collection

template<typename T , std::size_t MaxElements>
using cljonic::Vector< T, MaxElements >::cljonic_collection = std::true_type

The Vector constructor returns an instance of Vector initialized with its arguments. If the number of arguments exceeds the maximum number of elements specified, the extra arguments are silently ignored.

#include <iostream>
#include "cljonic.hpp"
using namespace cljonic;
int main()
{
auto v0{Vector<int, 10>{}}; // v0 is empty
auto v1{Vector<int, 10>{1, 2, 3, 4}}; // v1 is sparse
auto v2{Vector<int, 4>{1, 2, 3, 4}}; // v2 is full
auto v3{Vector<int, 4>{1, 2, 3, 4, 5, 6}}; // v3 is full, but the values 5 and 6 are ignored
auto v4{Vector{1, 2, 3, 4}}; // v4 is full of four int values
return 0;
}
