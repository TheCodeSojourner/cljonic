Detailed Description

template<SizeType MaxElements>
class cljonic::String< MaxElements >

The String type is a fundamental cljonic collection type. It is implemented as a C array of CharType, and does not use dynamic memory. A String has a specified maximum number of ordered character elements, which are always NUL terminated. A String is a function of its indexable elements. A String called with an out-of-bounds index will return its default element (i.e., a NUL). Many Core functions accept String arguments.
