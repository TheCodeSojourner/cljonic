Detailed Description

template<typename C>
class cljonic::core::CycleCollection< C >

CycleCollection The CycleCollection type implements as a lazy sequence composed of the values returned from a function of one parameter that is an index into an ordered collection of a specified length. The CycleCollection type cycles through the ordered collection over and over again repeatedly providing the elements of the ordered collection. Many Core functions accept CycleCollection arguments.
