Detailed Description

template<ValidCljonicContainerElementType T, SizeType MaxElements>
class cljonic::Set< T, MaxElements >

The Set type is a fundamental immutable collection type in cljonic. It is implemented as a C array, and does not use dynamic memory. A Set has a specified maximum number of unique unordered elements each of the same specified type (i.e., It is homogenous). Note that one could create a Set of a UNION to get something like heterogeneity. A Set is a function of its elements. A Set called with a value not contained within the set will return its default element. Many Core functions accept Set arguments.

#include "cljonic.hpp"

using namespace cljonic;

int main()
{
    const auto s0{Set<int, 10>{}};                // immutable, empty
    const auto s1{Set<int, 10>{1, 2, 3, 4}};      // immutable, sparse
    const auto s2{Set<int, 4>{1, 2, 3, 4}};       // immutable, full
    const auto s5{Set{1, 2, 3, 4}};               // immutable, full of four int values
    const auto s6{Set{1, 2, 1, 4}};               // immutable, sparse with three unique int values
