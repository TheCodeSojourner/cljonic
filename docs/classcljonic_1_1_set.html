Detailed Description

template<typename T, std::size_t MaxElements>
class cljonic::Set< T, MaxElements >

The Set type is a fundamental collection type in cljonic. It is implemented as a C array, and does not use dynamic memory. A Set has a specified maximum number of unique unordered elements each of the same specified type (i.e., It is homogenous). Note that one could create a Set of a UNION or std::variant to get something like heterogeneity. Many Core functions accept Set arguments.

◆ cljonic_collection

template<typename T , std::size_t MaxElements>
using cljonic::Set< T, MaxElements >::cljonic_collection = std::true_type

The Set constructor returns an instance of Set initialized with the unique elements in its arguments. If the number of unique elements in its arguments exceeds the maximum number of elements, the extra unique elements in its arguments are silently ignored.

#include "cljonic.hpp"
using namespace cljonic;
int main()
{
const auto s0{Set<int, 10>{}}; // s0 is immutable and empty
const auto s1{Set<int, 10>{1, 2, 3, 4}}; // s1 is immutable and sparse
const auto s2{Set<int, 4>{1, 2, 3, 4}}; // s2 is immutable and full
const auto s3{Set<int, 4>{1, 2, 3, 4, 5, 6}}; // s3 is immutable and full, but 5 and 6 are ignored
const auto s4{Set<int, 4>{1, 2, 1, 4, 5, 6}}; // s4 is immutable and full with 1, 2, 4, 5
const auto s5{Set{1, 2, 3, 4}}; // s5 is immutable and full of four int values
const auto s6{Set{1, 2, 1, 4}}; // s6 is immutable and full of three unique int values
return 0;
}
